# Mi Proyecto en TypeScript

Este proyecto es una aplicación escrita en TypeScript. A continuación se describen los archivos y su propósito.

## Estructura del Proyecto

```
mi-proyecto-ts
├── src
│   ├── index.ts          # Punto de entrada de la aplicación
├── package.json          # Configuración de npm y dependencias
├── tsconfig.json         # Configuración del compilador TypeScript
└── README.md             # Documentación del proyecto
```

## Instalación

Para instalar las dependencias del proyecto, ejecuta el siguiente comando en la raíz del proyecto:

```
npm install
```

## Compilación

Para compilar el proyecto, utiliza el siguiente comando:

```
npx tsc
```

## Ejecución

Una vez compilado, puedes ejecutar la aplicación con el siguiente comando:

```
node dist/index.js
```
